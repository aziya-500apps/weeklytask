Local Storage and Session Storage

Local Storage:-

The data is stored locally in system. The data can be avaliable till years.If we delete it then only it gets deleted.And Shared between all tabs and windows from the same origin
The data does not expire .It remains after the browser restart and even Os reboot.

Session Storage:-

For session storage we can keep expire days when the session gets expired(when expire time comes). The session storage exist only within the current browser tab. Another tab with the same pages will have a different storage but its shared between iframes in the same tab
